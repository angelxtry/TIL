# String Interpolation

## reference

https://www.programiz.com/python-programming/string-interpolation

## 정의

문자열의 placeholder를 변수의 값으로 대체하는 것.

## 방식

- %-formatting
- format() method
- f-strings
- ...

- %-formatting
  - 가장 오래됨
  - 많이 쓰임
- format() method
  - 많이 쓰임
- f-strings
  - 3.6에 도입됨
  - 편함(easy to use and increase the code readability)
