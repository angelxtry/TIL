# mysql 기본 명령어

## 접속

```cmd
mysql -u root -p
```

## Database 조회

```sql
show databases;
```

## Database 사용(선택?)

```sql
use [database_name];
```

## Database 삭제

```sql
show databases;
drop database [name];
show databases;
```

## table 조회

```sql
show tables;
```

## table 삭제

``` sql
DROP TABLE [table name];
```
