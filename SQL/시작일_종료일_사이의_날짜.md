# 시작일 종료일 사이의 날짜 가져오기

```sql
SELECT
TO_CHAR(TO_DATE(SD, 'YYYYMMDD') + (LEVEL -1), 'YYYYMMDD') T0
, TO_CHAR(TO_DATE(SD, 'YYYYMMDD') + (LEVEL), 'YYYYMMDD') T1
, TO_CHAR(TO_DATE(SD, 'YYYYMMDD') + (LEVEL +1), 'YYYYMMDD') T2
FROM (
   SELECT
         '20180505' SD
       , '20180508' ED
   FROM DUAL
   )
CONNECT BY TO_DATE(SD, 'YYYYMMDD') + (LEVEL -1) <= TO_DATE(ED, 'YYYYMMDD')
```
